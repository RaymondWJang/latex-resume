# LATEX-RESUME

This library generates a minimal, classic design resume based on Latex.

Currently has an error that has yet to be debugged. Might be an issue with the local Latex installation.


Structure:
- 📂 **src/** - Source code for the project.
  - 📄 ***.tex** - Subsections containing the contents of resume.
- 📂 **outputs/** - All final outputs after running resume.tex.
  - 📄 ***.pdf** - Final resume files
- 📂 **docs/** - Documentation files for the project.
  - 📄 Helpful documents for building a resume
- 📄 **.gitignore** - Specifies intentionally untracked files to ignore by Git.
- 📄 **resume.tex** - Tex file outlining the overall structure of the resume
